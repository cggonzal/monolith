Monolith Guides

Getting Started

Learn how to launch a basic blog with Monolith.

Installation

Clone the repository and start the development server:

$ git clone <repo> && cd monolith
$ make # or 'make run' if air is not installed

Visit http://localhost:9000 in your browser. The first run creates app.db and migrates the schema.

Creating a blog

Use the resource generator to scaffold posts:

$ make generator resource post title:string body:text

This generates a Post model, a controller with CRUD actions, templates and routes under /posts.

Run the server and you have a simple blog where you can create, edit and delete posts.
